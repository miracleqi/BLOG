# 背景

前端需要对输入的 IPV4 和 IPV6 进行格式验证，共有 IPV4、IPV6 和子网前缀的 6 中组合。

# 表达式

## IPV4
```
^((25[0-5]|2[0-4]\d|[01]?\d\d?)\.){3}(25[0-5]|2[0-4]\d|[01]?\d\d?)$
```

# 多行匹配

上面给出的都是单行正则匹配  
同事又提出，想要多行正则匹配（下面例子中换行符用 \n 表示）  
若正则表达式为  
```
^a$|^b$
```

则等价于

```
^((a)|(b))$
```

多行正则匹配为

```
^(((a)|(b))[\n])*((a)|(b))$
```
